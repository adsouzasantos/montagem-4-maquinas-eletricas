Dados de Entrada
Método dos Dois Wattímetros: Insira as leituras dos wattímetros W1 e W2, tensão de linha e frequência para calcular as potências trifásicas.
Fórmulas Utilizadas:
P₃φ = W1 + W2
Q₃φ = √3 × (W2 - W1)
S₃φ = √(P₃φ² + Q₃φ²)
FP = cos(φ) = P₃φ / S₃φ
C = Qc₃φ / (3 × 2π × f × VL²)
